# SentimentScope
SentimentScope Ã¨ una web platform per l'analisi del sentiment delle discussioni onlline utilizzando Reddit come fonte principale

Il sistema raccoglie i dati grazie ad un'API di Reddit, analizza il sentiment attraverso un algoritmo, salva i dati in un SQL Server database, e visualizza i dati e le statistiche attraverso una Dashboard Web.

## Features

- Ottieni i posts da Reddit utilizzando l'API ufficiale
- Calcola il sentiment di un post (positivo / neutrale / negativo)
- Raccoglie i dati ottenuti in un database
- Dashboard web interattiva con grafici
- Ricerca attraverso parole chiave e analisi nel tempo
